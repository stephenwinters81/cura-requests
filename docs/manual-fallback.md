---

### STEP 2: Complete a Paper Imaging Request Form

1. Take a **blank imaging request form** from the tray at **[LOCATION, e.g., printer room / reception desk / each consulting room]**.
2. Fill in all fields:
   - Patient name, DOB, phone, Medicare number, address
   - Referring doctor name and provider number
   - Exam type and clinical notes
   - Radiology practice name and fax number
3. Have the **referring doctor sign** the form.

---

### STEP 3: Call the Practice to Confirm the Booking

- Phone the radiology practice (numbers in Step 1).
- Confirm they can accept the patient and the requested exam.
- Note any booking reference number on the form.

---

### STEP 4: Fax the Request

1. Go to the clinic fax machine at **[LOCATION, e.g., admin office / printer room]**.
2. Clinic fax number (for cover sheet return number): **[CLINIC FAX NUMBER]**.
3. Place the completed form face-down in the feeder.
4. Dial the radiology practice fax number.
5. Wait for the confirmation printout showing the fax was sent successfully.
6. If the fax fails, try again. If it continues to fail, email the form as a scanned PDF to the practice email address (if available).

---

### STEP 5: File the Paper Copy

- Place the **original signed form** in the patient's physical chart.
- If no physical chart exists, file in the **"Pending Imaging Requests"** folder at **[LOCATION]**.

---

### STEP 6: Enter Retrospectively When the System is Restored

Once the Requests system is back online:

1. Log in to **requests.curaspecialists.com.au**.
2. Create a new request for each paper form submitted during the outage.
3. In the clinical notes field, add: **"Entered retrospectively -- originally submitted manually on [DATE]."**
4. Mark the paper form as entered (stamp or tick box).
